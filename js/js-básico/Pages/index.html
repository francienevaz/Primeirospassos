Laços de Repetição

Laços de Repetição é como traduzimos os Loops que usamos quando queremos repetir uma sequência de ações/comandos. Usamos uma variável que vai determinar a quantidade de vezes que aquela ação vai se repetir:

var numero = 5; for (var i = 0; i <= numero; i++) { console.log(" Executando o for" + i) }; console.log("Acabou!");

Nesse exemplo o "i" é o número de vezes que a ação vai se repetir. Cada vez que o 'laço for' passar/girar para verificar se i <= a váriavel 'numero' a variável "i" vai receber o número que representa a quantidade de vezes que aquele comando aconteceu.
