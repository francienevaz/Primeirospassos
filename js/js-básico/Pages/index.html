Laços de Repetição

Laços de Repetição é como traduzimos os Loops que usamos quando queremos repetir uma sequência de ações/comandos. Usamos uma variável que vai determinar a quantidade de vezes que aquela ação vai se repetir:

Nesse exemplo o "i" é o número de vezes que a ação vai se repetir. Cada vez que o 'laço for' passar/girar para verificar se i <= a váriavel 'numero' a variável "i" vai receber o número que representa a quantidade de vezes que aquele comando aconteceu. Outro exemplo de laço de repetição é o "While" que funciona de forma parecida com o "IF".

Uma observação para qual você deve se atentar é não esquecer de colocar o "++" para a variável que irá fazer o loop, pois se não, o seu programa entrará em loop infinito.
